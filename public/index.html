StockUp

A cryptocurrency trading bot

StockUp is an automated cryptocurrency trading bot created using Python. By scraping and analyzing live data online, StockUp is able to effectively flip BitCoin.

StockUp collects cryptocurrency data and stores it. Using that data, StockUp decides when is the best time to buy or sell.

StockUp is configured to trade only BitCoin at the moment. We plan to modify the bot to trade any cryptocurrencies in the future.
